# Design System Coverage

O Design System conta com **42 primitivas implementadas**.

## Cobertura nas Views Atuais
- `Button` e `Input`: 90% (alguns formulários antigos ainda têm inputs soltos).
- `Typography`: 95%.
- `Surface/GlassSurface/GlowSurface`: 80% (o Dashboard já as utiliza extensamente).
- Skeletons: 100% de padronização preparada, porém Views antigas (Grupos) ainda utilizam `Loading...` em texto.

A fundação é robusta, a falha reside nos componentes legados que ainda não foram substituídos para adotá-la.
